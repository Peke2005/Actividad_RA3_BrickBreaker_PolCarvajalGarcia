# Actividad RA3 - Juego Flame

Proyecto final de la practica Flutter + Flame.

## Ejecutar

```powershell
flutter pub get
flutter run -d chrome
```

## Verificar

```powershell
flutter analyze
flutter build web
```

## Entrega

- Proyecto: esta carpeta `actividad_ra3_juego_flame`.
- Documento: `docs/Documento_Actividad_RA3.md`.
- Capturas: `docs/capturas/`.

## Cambios sobre codelab

- Fondo arcade con estrellas y rejilla.
- Sprite propio del jugador dibujado en Canvas.
- Bola y ladrillos con estilo neon.
- Regla nueva: 3 vidas antes de perder.
- Puntos por fila: los ladrillos superiores valen mas.
